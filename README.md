# Number Guessing Game

A fun and simple Python game where the computer picks a random number and the player tries to guess it. The game gives hints if your guess is too high or too low, keeps track of your attempts, and lets you play multiple rounds.

## Features
- Random number selection between 1 and 100
- Hints for higher or lower guesses
- Tracks the number of attempts
- Replay option for multiple rounds
- Handles invalid input gracefully

## How to Play
1. Run the game using Python or Jupyter Notebook.
2. The computer will choose a number between 1 and 100.
3. Enter your guess when prompted.
4. The game will tell you if your guess is too high, too low, or correct.
5. Keep guessing until you find the correct number.
6. After winning, you can choose to play again or exit.
